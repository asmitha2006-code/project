#Daily Study Planner
####Video demo:
####Description:

A command-line based Daily Study Planner built using Python that helps users organize tasks, set priorities, track completion, and monitor daily progress.
All data is stored locally using a JSON file, making the application lightweight and easy to use.

This planner allows users to:
***Add daily tasks
Set task priorities(high,low,medium)
Mark tasks as completed
Delete tasks that are not necessary
Track daily progress 
Persistent data storage using JSON

Technologies/tools used:

Python 3
JSON for data storage
datetime module
Pytest for testing

Project Structure:

project.py
study_planner.json
README.md
test_project.py

Concepts used:

File handling
Dictionaries and lists
JSON serialization
Functions and modular programming

How to run the project:

1. Clone the repository
2. Navigate to the project folder
3. Run: python study_planner.py
